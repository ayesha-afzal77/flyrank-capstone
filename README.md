# FlyRank Capstone

A clean, responsive Flight Ranking and Aviation Analytics Dashboard.

## Tech Stack
- HTML5 (Semantic Elements)
- CSS3 (Flexbox & Grid Layout)
- Vanilla JavaScript

## How to Run
1. Clone this repository to your local system.
2. Open the folder in **Cursor IDE**.
3. Use the **Live Server** extension to launch the app in your browser.
4. Open your browser console (F12) to verify system logs.
